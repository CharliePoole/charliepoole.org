C# Source Code

ExpandingLabel.cs
A special type of label which can display a tooltip-like window to show the full extent of any text which doesn't fit. The window may be placed directly over the label or immediately beneath it and will expand to fit in a horizontal, vertical or both directions as needed.

FolderBrowser.cs
Wrapper class for the Win32 SHBrowseForFolder function. Allows the user to select a particular folder or create a new one.

WaitCursor.cs
Utility class used to display a wait cursor while a long operation takes place and guarantee that it will be removed on exit.
